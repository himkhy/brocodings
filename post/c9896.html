ពន្យល់ អត្តន័យនៃ Source Code

  • #include<stdio.h> : វាគឺជា Standard Input/output ដែលផ្ទុកនូវបណ្តា Function ដូច ជា Printf  មានន័យអាចប្រើ Prinf បានលុះត្រាតែមាន #include<stdio.h> នេះ ឬក៍គេហៅថាជា  Header file ក៍បានដែរ
  • #include<conio.h> : វាគឺជា Function ក្នុងការផ្ទុក Getch(); អាចប្រើPrinf បានលុះត្រាតែមាន #include<stdio.h>
  •  main(){} : វាគឺជា Main Fucntion ជាចំនុចដំណើរការ Code នៅក្នុង Program ទាំងអស់
  •  Printf : ជាការបង្ហាញទិន្នន័យចេញមកក្រៅលើ Command CMD
  • Getch(): សំរាបបញ្ឈប់ដំណើរការ Code ពេលដែល Press Anykey វានឹងបន្តដំណើរការទៀត
