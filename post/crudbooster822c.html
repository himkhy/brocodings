ការចាប់ផ្តើមជាមួយ CRUDBooster

CRUDBooster គឺជា CRUD Generator Package For Laravel​សម្រាប់បង្កើត CRUD បានលឿន (C​: Create, R: Read, U: Update, D: Delete or Distroy)។ CRUDBooster​ អាចឲ្យអ្នកបង្កើត​ AdminPage និង User Level មានសុវត្តិភាពងាយស្រួលកែ។

CRUDBooster គឺជា CRUD Generator for laravel, ជាមួយនិងលក្ខណៈពិសេសជាច្រើននៅក្នុង web application development។ វាងាយស្រួល easy, flexible, and powerful។

តម្លើង Installation Laravel ​5.4*

ដំបូងយើងត្រូវតម្លើង Composer 

ដំបូងយើងត្រូវតម្លើងLaravel ​5.4*

composer create-project --prefer-dist laravel/laravel blog "5.4.*"

Local Development Server

អ្នកប្រើប្រាស់ serve Artisan command. command និងចាប់ផ្តើមដំណើរការ server នៅក្នុង http://localhost:8000:

php artisan serve

តម្លើង​ Xampp Server

System Requirement and Basic Technical Knowledge

  • Web Server as:
    • Apache 2.4.x or higher with rewrite engine on (mod_rewrite)
    • Nginx 1.11.x or higher
  • Database that laravel supports, actually can be:
    • MySQL
    • Postgres
    • SQLite
    • SQL Server
  • Composer
  • Laravel 5.3.x or higher
  • Php 5.6.x or higher and the extensions:
    • Mcrypt
    • OpenSSL
    • Mbstring
    • Tokenizer
    • FileInfo

តម្លើង Installation

  1. បើក terminal, navigate to your project directory.
$ composer require crocodicstudio/crudbooster=5.4.*
  1. Add the following class, to "providers" array in the file config/app.php
crocodicstudio\crudbooster\CRUDBoosterServiceProvider::class,
  1. As outlined in the Migrations guide to fix this all you have to do is edit your AppServiceProvider.php file and inside the boot method set a default string length:
                use Illuminate\Support\Facades\Schema;
              
                public function boot(){
                  
                              Schema::defaultStringLength(191);
             
                 }

news detail from reference

4.Setting the database configuration, open .env file at project root directory

DB_DATABASE=**your_db_name**
DB_USERNAME=**your_db_user**
DB_PASSWORD=**password**

5. Run the following command at the terminal

$ php artisan crudbooster:install
