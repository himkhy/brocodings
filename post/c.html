មេរៀនទី ១ ៖ ការណែនាំស្គាល់ពី C PROGRAMMING

C PROGRAMMING គឺជា Program Language មួយដែលប្រើប្រាស់សំរាប់សរសេក្នុងការបង្កើតនូវសុំនុំ ពាក្យមួយចំនួន ដើម្បីធ្វើការជាលាក់ណាមួយ

មេរៀនទី ២ ៖ អថេ និងប្រភេទទិន្ន័យ

ប្រភេទទិន្នន័យត្រូវបានប្រើប្រាស់ ដើម្បីកមកំណត់តម្លៃរបស់អថេរ គេប្រើនប្រើនៅពេលប្រកាស់អថេរ

មេរៀនទី ៣ ៖ Control Structure

បើសិន Expression ពិត (Expression មានតម្លៃមិនសើ្មរសូន្យ ) Statement តូ្រវបានអនុវត្តន៏ តែផ្ទុយទៅវិញ បើ Expression មិនពិត ( Expression មានតម្លៃស្មើរសូន្យ ) វានឹងអនុវត្តន៏ Statement 2

មេរៀនទី ៤ ៖ Loop Control Structure

នោក្នុងផ្នែកនេះយើង នឹងបង្ហាញពីរដំណើរការ Repeat Statement ឬ មួយ Block Statement ដោយប្រើ លក្ខណៈអី្វមួយ ហៅថា Loop

មេរៀនទី ៥ ៖ Function

ដើម្បីដោះស្រាយបង្ហាញធំ ហើយស្មុគស្មាញយើងតូ្រវបំបែកបញ្ហាឲ្យទៅជាបង្ហាញតូចៗជាងមុនសិនទើបយើងអាចដោះស្រាយបង្ហាញធំៗនោះបាន

មេរៀនទី ៦ ៖ Array

Array គឺជាសុំនុំនៃ Variable ដែលមាន Data type ដូចគ្នា នឹងមានឈ្មោះតែមួយ។ ធាតុនីមួយៗរបស់ Array គឺ Variable 1 បើមានការកែប្រែនៅលើធាតុនោះ ធាតុដ៏ទៃគ្មានប្រែបួ្រលឡើយ

មេរៀនទី ៧ ៖ Pointer

Pointer គឺជាប្រភេទទិនិ្នន័យមួយ ដែលប្រាស្រ័យទៅលើប្រភេទទិនិ្នន័យដើម ដែលមានតម្លៃជា Address ចំពោះអញ្ញាតដែលយើងប្រើធ្វើការប្រកាសជា Pointer

មេរៀនទី ៨ ៖ File
