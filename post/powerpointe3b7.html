មេរៀនទី ១ ៖ ការណែនាំកម្មវិធី Microsoft Office PowerPoint

និយមន័យ

 Microsoft Office PowerPoint 2016 គឺជាកម្មវិធីមួយដែលសម្រាប់ប្រើនៅក្នុងការិយាល័យរដ្នបាល ឬកន្លែងដទៃទៀត ដើម្បីជួយសម្រយលដល់ការសរសេរអត្ថបទ ឬបង្កើតជារូបភាពផ្សេងៗ ដើម្បីបង្ហាញទៅលើផ្ទាំងកំណាត់ស សម្រាប់ធ្វើការពន្យល់ នែនាំ បង្ហាតបង្ហាញ ធ្វើការបង្រៀន ឬការងារដែលទាក់ទង់ និងការធ្វើបទបង្ហាញ។ ភាគច្រើនទិន្នន័យត្រូវរៀបចំជាមួយ​ និង Microsoft Office PowerPoint 2016 ហើយភា្ជប់ Computer ជាមួយនិង LCD Projector ដើម្បីបញ្ជាំងវាទៅលើផ្ទាំងកំណាត់ស ឬក្តារខៀនស ដើម្បីធ្វើ Presentation ។

របៀបបើកកម្មវិធី


យើងអាចបើកកម្មវិធី Microsoft Office PowerPoint 2016 តាមវិធីជាច្រើនដុចជា៖

របៀបទី១ ចុច Start Buttom -> ២. ចុច Search Programming -> ៣. ចុច -> Microsoft Office PowerPoint 2016

សិក្សាអំពីផ្ទាំង Interface របស់កម្មវិធី


មុននឹងឈានទៅដល់ការប្រើប្រាស់ យើងត្រូវតែស្គាស់ពីឈ្មោះរបស់

   Title Bar : សម្រាប់បង្ហាញអំពីឈ្មោះរបស់ File (Presentation) និងឈ្មោះរបស់កម្នវិធី (Microsoft​ Office PowerPoint)។

  Close Buttom : សម្រាប់ការចាកចេញពីកម្មវិធី Microsoft​ Office PowerPoint 2016

   Tabs : សម្រាប់ផ្ទុក និងបែងចែកនូវ Buttons សម្រាប់ប្រើប្រាស់ទៅតាបឲមមុខងាររៀងខ្លួន

   Grouup សម្រាប់ប្រមូល Buttons ផ្តំជាក្រុមៗ

Slide : ជាកន្លែងដែលត្រូវសរសេរអក្សរ ដាក់រូបភាពផ្សេងៗ ដើម្បីបង្ហាញក្នុងពេលធ្វើ Presentation

   Zoom Control : សម្រាប់ពង្រីក ឬបង្រួមផ្ទៃការងាររបស់ Slide

របៀបបើកកម្មវិធី

        ខាងក្រោមយើង នឹងបង្ហាញ Presentation ថ្មីមួយដូច្នេះសូមអនិវត្តតាមជំហានខាងក្រោម៖

របៀបទី ចុច File -> new-> Blank Presentation

របៀប Save File ទុក

គោលបំណងរបស់វាគឺដើម្បីរក្សាទុកនូវឯកសារដែលយើងប្រើប្រាស់ទុកក្នុង hard disk របស់ Computer  ឬក្នុង USB Flash Drive ។ ខាងក្រោមយើង នឹងថ្វើការ Save File មួយដោយដាក់ឈ្មោះថា File.pptx និង Save វាទុកនៅក្នុង Drive D របស់ Computer ។ ដូច្នេះសូមអនិវត្តតាមជំហានដូចខាងក្រោម៖
