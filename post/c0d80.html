Increment and Decrement Operator : កម្មវិធី    C បានផ្តល់ឲ្យមានប្រមាណវីធីផ្សេអទៀតដែលប្រើ សំរាប់ធ្វើការកំនើន ( Increment ) និង បន្តថយ ( Decrementing ) តម្លៃរបស់អេថរ ។ ប្រមាណវីធី           Increment (++) និងបូកបន្តែមមួយ នៅពេលដែលធ្វើការគណនា ។ ​​ ប្រមាណវីធី          Decrement (--) និងបូកបន្តែមមួយ នៅពេលដែលធ្វើការគណនា ។ 

 ប្រមាណវីធីទាំងពីនេះត្រូវបានប្រើប្រាស់តែមួយអថេរប៉ុណ្ណោះ ។ គេមិនប្រើជាមួយ Constant ឬកន្សោមទេ ។ ចំពោះប្រមាណវីធីទាំងពីរនេះ យើងអាចប្រើនៅពីមុខអថេរ (       ++n)ក៏បាន ឬពីក្រោអថេរក៏បាន ( n++)។

បញ្ជាក់៖      -(++n) មានន័យថាកើនមុនពេលប្រើប្រាស់ ( Ex. n=5 , x=++n , ពេលនោះ x=6 ) 

 -(n++)មានន័យថាកើនក្រោយពេលប្រើប្រាស់ ( Ex. x=n++, x=5, n=6)

 

#include<stdio.h>  main ( ) {  int a = 5;  int b = 5; 

printf( “Value of a: %d and b: %d\n”, a, b );  printf( “Value of a: %d\n”, ++a );  printf( “Value of b: %d\n”, b++ );  getch();

return ( 0 ); 

}
