មេរៀនទី ៤ ៖ Loop Control Structure

នោក្នុងផ្នែកនេះយើង នឹងបង្ហាញពីរដំណើរការ Repeat Statement ឬ មួយ Block Statement ដោយប្រើ លក្ខណៈអី្វមួយ ហៅថា Loop ។ ក្នុងភាសារ C Loop មានបី ប្រភេទគឺ ៖

            ១. While Loop Statement

            ២. do_while Loop Statement

             ៣. For Loop Statement 

 Loop គឺជាក្រុមនៃ Instruction Computer មួយដែល សកម្មភាពរបស់វាដដែរៗ ដោយគោរពតាមអថេរ ពិសេសណាមួយនៅពេលដែល Control Variable នៅតែពិត ។

While Loop Statement    ជាដំបូង While Loop បានតួ្រតពិនិត្យមើល Expression បើសិនវាពិតនោះ Statement របស់ While Loop តូ្រវបានអនុវត្តន៏ (Execute) រួចវាក៏ទៅពិនិត្យមើល Expression ម្តងទៀត បើសិនជាវានៅតែពិត នោះ Statement ឬ Block Statement របស់វានៅតែ Execute ដដែរ រហូតដល់ Expression មិនពិត ។

៤.២ .Do-While- Loop Statement    ជាចំបូង do While Loop បាន Execute មួយ Statement ដែលនៅក្នុង Loop សិនមុននឹងវាតួ្រតពិនិត្យមើល Expression ។ ពេលវាតួ្រតពិនិត្យឃើញថា Expression នៅតែពិត នោះ Statement តូ្រវបាន Execute ម្ដងទៀតជាបន្តបនា្ទប់ រហូតដល់វាពិនិត្យឃើញថា Expression លែងពិត ។  

For Loop Statement    ទំរង For loop មានបីកន្សោម ។ កន្សោមទីមួយតូ្រវបានប្រើសំរាប់កំណត់តម្លៃ ដំបូង  កន្សោមទី ២ ប្រើសំរាប់ ផ្ទៀងផ្ទាត់ថាតើ Loop និងតូ្រវបានបន្តម្ដងទៀត ឬ ទេ ហើយកន្សោមទី ៣ សំរាប់ប្តរតម្លៃដើម្បី Execute ។

Goto Statement    ប្រើសំរាប់ចាក់ចេញ បញ្ចួល បង្វែលតួ្រតពិនិត្យ ទៅលើ Statement ណាមួយយ៉ាងឆាប់រហ័យ ។  ទំរងទូទៅនៃ Goto label
