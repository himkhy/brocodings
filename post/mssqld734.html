មេរៀនទី១៣ ៖ Built-in Function in SQL Server

  In SQL Server has a lot of Built-in Function that allow you to use that Built-in Function to create other User Defined Function.

  Built-in Function in SQL Server such as:

  - Conversion Function

  - Date Time Function

  - Logical Function

  - String Function

  - Ranking Function

a.Conversion Function

Cast(expression as datatype,): use to convert from Data Type to other Data Type.

Ex:

  Declare @Result int=5

  Select Cast(@Result as Float)/2;

Convert(datatype,expression): use to convert from Data Type to other Data Type.

Ex:

  Declare @Result int=5

  Select Convert(Float,@Result)/2;

Note: Format() : is a function use to format string or other data type to other style writing.

Ex: Select Format(12,'00.00'); it will return value 12.00.

b. DateTime Function

GETDATE(): get current date and time of system.

  Ex: Select GetDate() as Now; 2014-03-21 14:57:12.200

DAY(Expression): Get day of expression.

  Ex: Select Day(GetDate()) as Days

MONTH(Expression): Get month of expression.

  Ex: Select Month(GetDate()) as Month

YEAR(Expression): Get year of expression.

  Ex: Select Year(GetDate()) as Year

EOMONTH(Expression): Get last day in month of expression.

  Ex: Select EOMONTH(GetDate()) as [End OF Month]

DATEADD(Interval,Increment,DateTime): adds or subtracts a specified time interval from a date.

  Ex: Select DATEADD(DAY,2,GetDate()) as [Custome Date]; 2014-03-23 15:01:29.627

DATEDIFF(Interval,Start Date, End Date): Returns the count (signed integer) of the specified datepart boundaries crossed between the specified startdate and enddate.

  Ex: Select DATEDIFF(Day,GetDate(),Convert(Datetime,'03-25-2014'));

ISDATE(DateTime): Validate Field For DATE datatype.

   Ex: Select ISDate(GetDate()) as CheckDate;

DatePart(Interval,DateTime): used to return a single part of a date/time, such as year, month, day, hour, minute.

  Ex: Select DatePart(DAYOFYEAR,GetDate());

C. Logical Function

CHOOSE(index,val1,val2,val3,...)  the item at the specified index from a list of values in SQL Server 2012.

   Ex: SELECT CHOOSE (3, 'Manager', 'Director', 'Developer', 'Tester' ) AS Result;

  Result៖  Developer

IIF ( boolean_expression, true_value, false_value ) using for inline condition and we can use it in select statement.

  Ex:

  DECLARE @a int = 45, @b int = 40;

  SELECT IIF ( @a > @b, 'TRUE', 'FALSE' ) AS Result;

Result:

  TRUE

D. String Function

ASCII(char) : Returns the ASCII code value of the leftmost character of a character expression.

  Ex:   Select ASCII('A');  it return 65.

CHAR(int) : Converts an int ASCII code to a character.

   Ex:  Select CHAR(65);  it return A.


LEFT(string,int) : Returns the left part of a character string with the specified number of characters.

  Ex:   Declare @str Nvarchar(50)='Hello World'

  Select LEFT(@str,5)  

LEN(string) : Returns the number of characters of the specified string expression, excluding trailing blanks.

  Ex:  LEN(‘Hello World’);     it return 11.

REVERSE(string) : Returns the reverse order of a string value.

  Ex:  REVERSE(‘Hello World’);     it return ‘dlroW olleH’.

RIGHT(string,int) : Returns the right part of a character string with the specified number of characters.

  Ex:   Declare @FullName Nvarchar(50)=‘Sok Dara’

    Select Right(@FullName,4);  Dara

SPACE(int) : Returns a string of repeated spaces.

  Ex:  Declare @LastName Nvarchar(50)=‘Sok’

   Declare @FirstName Nvarchar(50)=‘Dara’

    Select Concat(@LastName,Space(8),@FirstName) As [Full Name] ;

 SUBSTRING(string,start,lenght)  : Returns part of a character, binary, text, or image expression in SQL Server 2012.

  Ex:  Declare @str Nvarchar(50)='Hello World'

  Select SubString(@str,7,5)

Lower(string) : Returns a character expression after converting uppercase character data to lowercase.

  Ex:  Select LOWER(‘Hello World’);     Result: hello world;

UPPER(string) : Returns a character expression with lowercase character data converted to uppercase.

  Ex:  Select UPPER(‘Hello World’);     Result: HELLO WORLD;

CharIndex(Expression,String[,Start]): Searches an expression for another expression and returns its starting position if found.

  Ex: SELECT CHARINDEX(' ','Hello World'); 


E. Ranking Function

  Rank : Returns the rank of each row within the partition of a result set. The rank of a row is one plus the number of ranks that come before the row in question.

  Dense Rank: Returns the rank of rows within the partition of a result set, without any gaps in the ranking. The rank of a row is one plus the number of distinct ranks that come before the row in question.

  Row_Number : Returns the sequential number of a row within a partition of a result set, starting at 1 for the first row in each partition.

      Ex:

SELECT p.FirstName, p.LastName

    ,ROW_NUMBER() OVER (ORDER BY a.PostalCode) AS "Row Number"

    ,RANK() OVER (ORDER BY a.PostalCode) AS Rank

    ,DENSE_RANK() OVER (ORDER BY a.PostalCode) AS "Dense Rank"

    ,s.SalesYTD

    ,a.PostalCode

FROM Sales.SalesPerson AS s

    INNER JOIN Person.Person AS p

        ON s.BusinessEntityID = p.BusinessEntityID

    INNER JOIN Person.Address AS a

        ON a.AddressID = p.BusinessEntityID
