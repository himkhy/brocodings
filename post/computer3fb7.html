CPU (Central Processing Unit)

CPU (Central Processing Unit)ជាកន្លែងសម្រាប់ដំណើរការនូវ Instruction របស់ Software ដើម្បីដំណើរការព័ត៌មាន។ ដែលយើងអាចនិយាយបានថាវាជាខួរក្បាលសម្រាប់ធ្វើការគណនា​រាល់ការបញ្ជានានា ហើយ បញ្ជូនទិន្នន័យទៅកាន់ Monitor វិញ ។​ វាត្រូវបានគេចែកចេញជា ០២ ផ្នែកគឺ ៖

-           Control Unit

-           Arithmetic Logical Unit

Control Unit  វាមានតួនាទីប្រាប់ទៅសមាសភាពរបស់ប្រព័ន្ធកុំព្យូទ័រ ពីរបៀបនៃការ បញ្ជូនចេញរបស់ Instruction របស់កម្មវិធីមួយ ។ វាមានចលនានៃសញ្ញាអគ្គីសនីរវាង Main Memory និង​ Arithmetic Logical Unit ។ វាគឺជាសញ្ញាអគ្គីសនីផ្ទាល់រវាង Main Memory និង Input ហើយនឹងធ្វើការ Output ឬបង្ហាញផងដែរ ។​

Arithmetic Logical Unit ឬយើងអាចហៅម្យ៉ាងទៀតថាជា ALU ជាទម្រង់នៃដំណើរការ ប្រមាណវិធី និង ដំណើរការ Logical ហើយវាត្រូវបានត្រួតពិនិត្យល្បឿននៃដំណើរការទាំង នោះ ។
